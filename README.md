# 🤖 n8n AI-Powered Personal Dashboard

An automated personal assistant built with n8n that monitors, 
summarizes, and delivers information via Telegram.

## 🗺️ Project Phases

| Phase | Description | Status |
|-------|-------------|--------|
| Phase 1 | Telegram Bot + Morning Briefing | ✅ Complete |
| Phase 2 | Gmail AI Summarizer | 🔄 In Progress |
| Phase 3 | GitHub Tracker + Notion Logger | ⏳ Upcoming |
| Phase 4 | Full Dashboard + Documentation | ⏳ Upcoming |

## 🛠️ Built With
- [n8n](https://n8n.io) — Workflow automation
- Telegram Bot API
- OpenAI (Phase 2+)
- Gmail API (Phase 2+)
- GitHub API (Phase 3+)
- Notion API (Phase 3+)
